PRANA YOGA

2024   |   UI/UX Design   |   Wellness Platform / Yoga Website

Overview

Prana is a yoga class booking platform designed to create a seamless and calming digital experience for users exploring wellness and online yoga sessions. The platform enables users to discover instructors, book classes, manage schedules, and access personalized wellness services through an intuitive interface.

The goal of this project was to simplify the booking journey while maintaining a visually balanced and user-friendly experience that reflects the mindfulness and simplicity associated with modern wellness platforms.

My Role

I worked on the complete UI/UX design process for the platform — focusing on user flows, booking interactions, dashboard structuring, and high-fidelity interface design. The objective was to create a clean, accessible, and engaging experience that supports effortless class discovery and session management.

  • User flow and booking experience design
  • Wireframing and interface structuring
  • High-fidelity mobile and web UI design
  • Dashboard and schedule management UX
  • Wellness-focused visual design system
  • Responsive and user-centered experience planning
